# Ash Mail

A temporary email web app made with the free api of [mail.tm.](https://mail.tm)

## Made using

- [Svelte Kit](https://kit.svelte.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Daisy Ui](https://daisyui.com/)

## Running locally

```bash
# Installing all dependencies
npm install

# Running the web app
npm run dev -- --open
```
